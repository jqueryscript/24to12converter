군사 시간이란?

군사 시간은 24시간제를 사용하는 시간 표기법으로, 오전과 오후를 구분하지 않습니다. 이 시스템은 군대, 항공, 응급 서비스에서 정확성을 위해 사용됩니다.

  • • 하루를 24시간으로 표시 (0000-2359)
  • • AM/PM 구분이 필요 없음
  • • 국제적으로 표준화됨
  • • 오류 가능성 감소

군사 시간 읽는 법

군사 시간은 네 자리 숫자로 읽으며, 처음 두 자리는 시간, 마지막 두 자리는 분을 나타냅니다.

  • • 1430 = "십사 서른" (2:30 PM)
  • • 0800 = "제로 에이트 헌드레드" (8:00 AM)
  • • 0000 = "제로 백" (자정)
  • • 2400 = "투블러 헌드레드" (자정)
