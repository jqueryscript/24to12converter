旅行のための時間換算表

2026年6月24日更新

アメリカからヨーロッパ、中南米、アジアに旅行する場合、目にするすべてのスケジュールは24時間表示を使っています。変換するには、13〜23時から12を引いてPMを付けるだけです。このガイドでは、印刷可能なポケットチャートと、フライト、鉄道、ホテルの案内、レストランの確認画面で実際に目にする正確な時間を紹介します。

旅行者のための1つの変換ルール

時間が13以上の場合は12を引くだけです。14:00 = 午後2時。19:30 = 午後7時30分。22:00 = 午後10時。時間が00の場合は深夜です。01〜11の場合はAMです。分は変わりません。旅行中に時間を再確認したいときは、スマートフォンで変換ツールを開いて確認してください。

観光地・博物館

開館時間の案内板と入場券

Ouvert 10:00〜18:00 = 午前10時〜午後6時開館。最終入場は閉館の1時間前が一般的です。フランスやイタリアでは月曜休館が多いです。

その場で素早く変換する3つのコツ

コツ1：2を引いて、最初の数字を消す。14:00：14 - 2 = 12、1を消す = 午後2時。17:00：17 - 2 = 15、1を消す = 午後5時。21:00：21 - 2 = 19、1を消す = 午後9時。PMの時間にのみ使えます。

コツ2：午後のブロックを暗記する。13 = 午後1時、14 = 午後2時、15 = 午後3時、16 = 午後4時、17 = 午後5時。この5つを覚えれば、あとはパターンを拡張するだけです。18 = 6、19 = 7、20 = 8、21 = 9、22 = 10、23 = 11。

コツ3：スマートフォンを24時間表示に設定する。旅行前に、iPhone（設定 > 一般 > 日付と時刻 > 24時間表示）またはAndroid（設定 > システム > 日付と時刻 > 24時間形式を使用）を24時間表示に切り替えましょう。ロック画面に14:00と表示されるのを2日も見れば、変換は自動的に身につきます。

クイック参照：13:00 = 午後1時、14:00 = 午後2時、15:00 = 午後3時、16:00 = 午後4時、17:00 = 午後5時、18:00 = 午後6時、19:00 = 午後7時、20:00 = 午後8時、21:00 = 午後9時、22:00 = 午後10時、23:00 = 午後11時。
