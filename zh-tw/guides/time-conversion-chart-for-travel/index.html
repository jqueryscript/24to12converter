三個快速轉換小技巧

技巧一：減2再去掉第一位數字。14:00：14 - 2 = 12，去掉1 = 下午2點。17:00：17 - 2 = 15，去掉1 = 下午5點。21:00：21 - 2 = 19，去掉1 = 晚上9點。僅適用於下午時間。

技巧二：記住下午區塊。13 = 下午1點，14 = 下午2點，15 = 下午3點，16 = 下午4點，17 = 下午5點。記住這五個之後，其餘只是延伸規律：18 = 6，19 = 7，20 = 8，21 = 9，22 = 10，23 = 11。

技巧三：將手機設為24小時制模式。出發前，將你的iPhone（設定 > 一般 > 日期與時間 > 24小時制）或Android（設定 > 系統 > 日期與時間 > 使用24小時制）改為24小時顯示。在鎖定螢幕上看到14:00兩天後，轉換就會變成直覺反應。
