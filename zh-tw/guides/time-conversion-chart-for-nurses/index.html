護理人員時間轉換表

更新日期：2026年6月20日

醫院採用24小時制是為了防止危險的AM/PM混淆。轉換24小時制只需將13至23的小時減去12並加上PM。本指南提供可列印的快速對照表、常見護理班表時間以及給藥時程，讓你快速在兩種格式之間切換。

為什麼24小時制在護理中至關重要

在給藥紀錄或醫囑上寫「2:00」有50%的混淆風險——是上午還是下午？24小時制消除了這種模糊性：02:00永遠是凌晨2點，14:00永遠是下午2點。這就是為什麼聯合委員會、WHO和ISMP都建議在藥物記錄中使用24小時制。使用轉換工具確認任何時間。

如何快速心算轉換

幾個班次之後，轉換就會變成直覺反應。以下多數護理人員使用的快速心算法：

下午時間（13:00–23:59）：小時減去2，再去掉第一位數字。14:00 → 14 − 2 = 12 → 去掉1 → 下午2:00。17:00 → 17 − 2 = 15 → 去掉1 → 下午5:00。原理其實就是減去12，結果一樣。

上午時間（00:00–11:59）：去掉前面的零加上AM即可。07:00 = 上午7:00。例外：00:00是上午12:00（午夜）。

中午：12:00是下午12:00。這個時間常讓人困惑，因為看起來像是上午。切記：24小時制中的12:00永遠是中午，絕非午夜。

練習參考：13:00 = 下午1點，14:00 = 下午2點，15:00 = 下午3點，16:00 = 下午4點，17:00 = 下午5點，18:00 = 下午6點，19:00 = 下午7點，20:00 = 下午8點，21:00 = 下午9點，22:00 = 下午10點，23:00 = 下午11點。

護理學生的記錄技巧

  • 務必使用四位數字。寫08:00而非8:00。前導零在病歷中很重要。
  • 午夜是00:00，不是24:00。多數電子病歷系統和給藥紀錄使用0000表示午夜。
  • 敘述性記錄用冒號（08:00），流程表省略冒號（0800）。兩種寫法都正確，請依機構規定。
  • 仔細檢查12的狀況。12:00 = 中午，00:00 = 午夜。這是護理記錄中最常見的錯誤。
  • 保持一致性。單位在給藥紀錄中使用24小時制，你的記錄也應使用相同格式。混用格式會造成混淆。
