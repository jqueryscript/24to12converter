Zeit-Addierer

Addieren oder subtrahieren Sie Stunden und Minuten von jeder Uhrzeit. Geben Sie eine Startzeit und den zu addierenden Betrag ein, und sehen Sie das Ergebnis sofort. Unterstützt 24-Stunden-Format und Mitternachtsübergang.

Ergebnis

15:30

(15:30 Uhr)

So verwenden Sie den Zeit-Addierer

1

Geben Sie eine Startzeit im 24-Stunden-Format ein (z.B. 14:00 für 14:00 Uhr).

2

Geben Sie die zu addierenden Stunden und Minuten ein. Verwenden Sie negative Zahlen, um Zeit zu subtrahieren.

3

Sehen Sie das Ergebnis sowohl im 24-Stunden- als auch im 12-Stunden-Format. Zeiten nach Mitternacht werden korrekt umgerechnet.
